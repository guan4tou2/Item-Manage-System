JSON 格式

JSON 陣列格式，每個物件代表一個物品：

[
  {
    "ItemID": "ITEM-001",
    "ItemName": "物品名稱",
    "ItemType": "電子產品",
    "ItemStorePlace": "客廳電視櫃",
    "ItemFloor": "1F",
    "ItemRoom": "客廳",
    "ItemZone": "電視櫃",
    "ItemOwner": "擁有者",
    "ItemGetDate": "2024-01-01",
    "WarrantyExpiry": "2025-01-01",
    "UsageExpiry": "",
    "ItemDesc": "描述內容"
  }
]
