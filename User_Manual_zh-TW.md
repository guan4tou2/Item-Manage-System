# 物品管理系統 使用手冊

歡迎使用 **物品管理系統 (Item Manage System)**！這是一套專為現代家庭與小型辦公室設計的物品追蹤解決方案。本手冊將引導您熟悉系統的各項功能。

---

## 🚀 快速入門

### 1. 登入與註冊
*   **首次使用**：請點擊登入頁面的「註冊新帳號」連結。
*   **登入**：輸入您的帳號密碼。系統支援「記住我」功能，下次可直接進入。
*   **忘記密碼**：點擊「忘記密碼？」，系統將提示您聯繫管理員重置。

### 2. 安裝應用程式 (PWA) ✨
本系統支援安裝為應用程式，讓您在手機上獲得最佳體驗：
*   **桌面端 (Chrome/Edge)**：點擊網址列右側的「安裝」圖示。
*   **iOS (Safari)**：點擊「分享」按鈕 -> 選擇「加入主畫面」。
*   **Android (Chrome)**：點擊選單 -> 選擇「安裝應用程式」。
> **特色**：安裝後可離線瀏覽部分快取內容，且開啟速度更快！

---

## 📦 物品管理

### 新增物品
1.  點擊導航欄的 **「新增物品」**。
2.  **上傳照片**：強烈建議上傳物品照片，系統會自動生成縮圖。
3.  **填寫資訊**：
    *   **名稱**：物品的識別名稱。
    *   **位置**：選擇樓層、房間與區域（如：1F -> 客廳 -> 電視櫃）。
    *   **期限**：設定「保固到期日」或「使用期限」，系統會自動在首頁發出警報提醒。
4.  點擊「新增」完成。

### 批量操作 (管理員專用) ✨
想要一次整理多個物品嗎？
1.  在首頁，滑鼠游標移動到物品卡片上。
2.  勾選您想要處理的物品左上角的 **Checkbox**。
3.  畫面下方會自動出現 **操作工具列**。
4.  您可以選擇：
    *   🚚 **批量移動**：一次將多個物品移動到新的儲存位置。
    *   🗑️ **批量刪除**：一次刪除多個物品（請謹慎操作！）。

### 條碼與 QR Code
每個物品都有專屬的識別碼：
*   **QR Code**：可列印貼, 手機掃描即可直接開啟物品詳情頁。
*   **條碼 (Barcode)**：支援一般大賣場條碼掃描槍快速盤點。

## 🔍 搜尋與篩選

### 智慧搜尋
首頁上方的搜尋框支援：
*   **關鍵字**：輸入名稱、描述或 ID。
*   **即時建議**：輸入 2 個字以上，系統會自動跳出建議選項。

### 進階篩選抽屜 ✨
點擊搜尋框旁的 **「篩選」** 按鈕，右側會滑出進階篩選面板：
*   **詳細位置**：精確篩選特定「樓層」、「房間」或「區域」的物品。
*   **類型**：只顯示特定分類（如：電子產品）。
*   **狀態**：快速找出「保固過期」或「即將過期」的物品。

---

## ⚙️ 進階功能

### 到期通知
點擊右上角的 **鈴鐺圖示 🔔**，即可查看：
*   🔴 **已過期**：保固或使用期限已過的物品。
*   🟡 **即將到期**：30 天內即將到期的物品。

### 暗色模式 (Dark Mode)
系統會自動偵測您的裝置設定。您也可以點擊導航欄的 **月亮/太陽圖示** 手動切換。

---

## ❓ 常見問題

**Q: 誤刪了物品可以救回嗎？**
A: 目前刪除操作是永久性的，建議刪除前再三確認。

**Q: 如何備份資料？**
A: 管理員可至「系統管理」->「備份與還原」下載完整的 JSON 備份檔。
