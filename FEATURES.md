# 📍 物品位置記錄系統

## 🎯 系統定位

這是一個專注於**記錄物品放置位置**的管理系統，讓您輕鬆記錄家中物品的具體位置，再也不怕找不到東西！

## ✨ 核心功能

### 📸 照片記錄
- **上傳物品照片** - 支援JPG, PNG, GIF格式
- **自動縮圖** - 提升載入速度
- **照片預覽** - 直觀查看物品外觀

### 📍 位置記錄
- **精確位置記錄** - 記錄樓層 / 房間 / 區域與完整放置地點
- **位置搜尋** - 按地點、樓層、房間、區域快速找到物品
- **位置更新** - 隨時更新物品的新位置
- **位置標籤** - 清楚顯示物品位置

### 🔍 智能搜尋
- **名稱搜尋** - 按物品名稱搜尋
- **地點搜尋** - 按放置地點 / 樓層 / 房間 / 區域搜尋
- **模糊匹配** - 支援部分關鍵字搜尋
- **排序** - 可依保固/使用期限、名稱排序

### 🛡 保固與期限
- **保固截止** - 紀錄保固期限
- **使用期限** - 紀錄使用/保存期限
- **到期提醒** - 顯示即將到期資訊
- **提醒梯度** - 可自訂多階段提醒天數
- **通知管道** - Email 及 Web Push/LINE/Telegram 預留

### 📦 庫存與補貨
- **庫存數量** - 記錄庫存與剩餘量
- **安全庫存** - 低於安全門檻顯示警告
- **補貨門檻** - 低於門檻列入補貨清單
- **補貨清單匯出** - CSV/JSON 下載

### 📊 統計分析
- **總物品數** - 記錄的物品總數
- **照片統計** - 有照片的物品數量
- **位置統計** - 有位置記錄的物品數量
- **分類統計** - 有分類的物品數量
- **保固統計** - 可依到期排序

### 📦 QR / 條碼
- **標籤產生** - 生成物品 QR 與條碼（PNG）
- **下載列印** - 直接下載標籤
- **掃描搜尋** - 前端相機掃描自動帶入搜尋

## 📱 操作流程

### 1. 新增物品
```
登入系統 → 點擊「新增物品」 → 填寫物品資訊 → 上傳照片 → 記錄位置 → 儲存
```

### 2. 搜尋物品
```
輸入物品名稱或地點 → 點擊搜尋 → 查看結果 → 找到物品位置
```

### 3. 更新位置
```
進入管理頁面 → 找到物品 → 輸入新位置 → 點擊更新 → 完成
```
